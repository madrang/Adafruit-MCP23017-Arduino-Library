This is a library for the MCP23017 I2c Port Expander
 
These chips use I2C to communicate, 2 pins required to interface

To download. click the DOWNLOADS button in the top right corner, rename the uncompressed folder MCP23017. Check that the MCP23017 folder contains MCP23017.cpp and MCP23017.h

Place the MCP23017 library folder your <arduinosketchfolder>/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE.
